# rems-client

This package contains an autogenerated HTTP/REST client for REMS.

## Regenerating the client

Prerequisites:

```sh
npm install -g swagger2openapi
go install github.com/oapi-codegen/oapi-codegen/v2/cmd/oapi-codegen@latest
```

Run the generation script from this directory:

```sh
./gen.sh
```

This will:

1. Download the live Swagger 2.0 spec from the configured REMS instance.
2. Convert it to OpenAPI 3.0 (`openapi3.json`) using `swagger2openapi`.
3. Generate the Go client (`rems.gen.go`) using `oapi-codegen`.

## Configuration

- The source URL is set in `gen.sh` (`SWAGGER_URL`).
- The `oapi-codegen` output options (package name, output file, etc.) are in `oapi-codegen.yaml`.
